
#####################
NCM\::Component\::ccm
#####################


****
NAME
****


The \ *ccm*\  component manages the configuration file
for CCM.


***********
DESCRIPTION
***********


The \ *ccm*\  component manages the configuration file for the CCM
daemon.  This is usually the `/etc/ccm.conf` file. See the ccm-fetch
manpage for more details.
